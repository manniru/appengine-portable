GAEPy Readme

Overwiew

Google Appengine Python SDK, launcher GUI and local Development Server packed as PortableApp.
Contains customized Python 2.5.4 distribution with ssl for certificate validating (and PIL, but untested).
  • GAE Launcher GUI Screenshots
Runs on Windows XP (and worse up) out of the package directory without dependencies.
Ready-to-Run example guestbook (local uses sqlite-db)

Installation

Download and run the portable installer "GAEPyPortable_*.paf.exe".
For online installers (executables matching "*_online.paf.exe") the binary distribution of
"Google Appengine SDK" and/or "Python for Windows" will be downloaded on installation.
For offline installation download and place the approriate *.7z archive into the same directory.

For first installation make shure to check "Initialize Configuration Files" in the installer dialog,
this will setup the path to pythonw.exe and configure sample guestbook application ready to run.

Running the GUI

Running the SDK Click on "GAEPyPortable.exe" in the application directory
(by default: "\PortableApps\GAEPyPortable").
In the launcher select the project "guestbook" and click on the icon "Run".
After some moments the appengine development server starts on http://localhost:8080
and the icon Browse will become active. Click to view the site in your browser.

Uninstall

Remove the directory "\PortableApps\GAEPyPortable" from your USB-drive
