= Google App Engine SDK Python Portable (GAEPy)

Google Appengine Python SDK, launcher GUI and local Development Server as Portable Application.
Application directory relocatable, no dependencies on other installed software.
Runs on Windows XP, Vista, 7 and later, Ready-to-Run example guestbook with local sqlite-db
