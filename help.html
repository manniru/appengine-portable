= Google Appengine Python Portable

Overwiew

Google Appengine Python Development Server packed as PortableApp.
Contains customized Python 2.5 distribution with SSL and PIL,
Runs on Windows XP (and worse) out of the package directory.
Ready-to-Run example guestbook (local uses sqlite-db)

Installation

Download and run the portable installer "GoogleAppenginePythonPortable_1.3.8_*.paf.exe".
For installers matching "*_online.paf.exe" binary distribution of "Google Appengine" and "Python for Windows" will be downloaded online as part of the installation.
For offline installation download and place the approriate *.7z archive into the same directory.
Appengine distribution has not been modified except compressed by PortableApps Compactor.
Python distribution with additional modules ssl and PIL.
Excluded modules: TCL/TK, Doc etc.

Running the SDK

Click on "GoogleAppenginePythonPortable.exe" in the application directory
(by default: "\PortableApps\GoogleAppenginePythonPortable").
In the launcher select the project "guestbook" and click on the icon "Run".
After some moments the appengine development server starts on http://localhost:8080
and the icon Browse will become active. Click to view the site in your browser.

Uninstall

Remove the directory "\PortableApps\GoogleAppenginePythonPortable" from your USB-drive.
