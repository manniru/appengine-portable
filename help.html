= Google Appengine Python Portable

Installation

Download and run the portable installer "GoogleAppenginePortable_1.3.7_online.paf.exe".
The binary distributions of "Google Appengine" and "Python for Windows" will be downloaded
online as part of the installation.

Running the SDK

Click on "GoogleAppenginePythonPortable.exe" in the application directory
(by default: "\PortableApps\GoogleAppenginePythonPortable").
In the launcher select the project "guestbook" and click on the icon "Run".
After some moments the appengine development server starts on http://localhost:8080
and the icon Browse will become active. Click to view the site in your browser.

Uninstall

Remove the directory "\PortableApps\GoogleAppenginePythonPortable" from your USB-drive.
